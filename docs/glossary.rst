
.. _glossary:

Glossary
========

.. glossary::

   bake-time
      The moment when the Bakefile tool ``bkl`` is run to *create* native
      makefiles and projects. Nothing is compiled at this time, only the
      makefiles are created. Compare with *make-time*.

   make-time
      The moment when a generated makefile, project or solution is *used*
      by the user to build their project. Compare with *bake-time*.
