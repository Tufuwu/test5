SIFT Documentation
==================

SIFT, Satellite Information Familiarization Tool, is a GUI application
for viewing and analyzing earth-observing satellite data. This documentation
includes information on high-level concepts and how-tos of SIFT, but low-level
information for advanced features and how to contribute to the project can also
be found.

For general information on the use of SIFT, official releases, and
installation instructions of those releases see the SIFT
`home page <http://sift.ssec.wisc.edu/>`_. Note that the library and
python package for the SIFT project are named ``uwsift``.

For source code see the `GitHub repository <https://github.com/ssec/sift>`_.
